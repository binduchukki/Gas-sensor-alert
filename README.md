# 🚨 Gas Sensor Alert System using Arduino

## 📌 Description
This project is a **Gas Sensor Alert System** using Arduino that detects harmful or flammable gases in the environment. It uses a **gas sensor (MQ-2/MQ-135)** to monitor air quality and alerts users when gas levels exceed a safe limit.

---

## 🔧 Features
- 🚨 Detects gas leakage in real time  
- 🔊 Buzzer alert when gas is detected  
- 🔴 LED indication for warning  
- ⚡ Continuous monitoring  
- 🔁 Fast response system  

---

## 🧰 Components Used
- Arduino Uno  
- Gas Sensor (MQ-2 or MQ-135)  
- Buzzer  
- LED  
- Resistor  
- Breadboard  
- Jumper Wires  

---

## ⚙️ Working Principle
1. Gas sensor detects gas concentration in the air  
2. Arduino reads analog values from the sensor  
3. If gas level exceeds threshold:
   - Buzzer turns ON  
   - LED turns ON  
4. If gas level is normal:
   - Buzzer OFF  
   - LED OFF
